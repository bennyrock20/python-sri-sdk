# SRI 

SRI provides a simple interface to the SRI API.

## Dependencies

- Python 3.6+
- pipenv
- requests
- json
- pytest

## Installation

Create a virtual environment and install the dependencies:

```bash
pipenv shell --three
```

Install the package:

```bash
pipenv install 
```

## Commands

Create a public key and private key:

```bash
python -m sri_sdk create_keys
```
